CS 360 – Weight Tracker App

Rebecca Ferjo's enhanced artifact for the CS 360 Mobile Architecture course.

📱 Project Overview

This mobile application was created in CS 360 to allow users to track and monitor their weight over time. The original app included basic functionality for adding and viewing weights using Android Studio and Java.

⚙️ Enhancements

The enhanced version introduced secure login/authentication features, error handling during user sign-in, and improved UI consistency across screens. Notifications were added to alert users when goals were achieved, and data validation was implemented for better user input control.
